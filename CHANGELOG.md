# TokenTracker 插件更新日志

## v1.2.0 - 2026-04-03

### 🚨 安全修复（最高优先级）

#### 并发安全问题修复
**问题**: `_get_session_lock` 方法存在竞态条件，多个线程可能同时创建不同的锁，导致并发控制失效。

**修复**: 使用 `dict.setdefault()` 原子操作替代原有的检查-创建模式。

**修复前**:
```python
def _get_session_lock(self, sid: str) -> asyncio.Lock:
    if sid not in self._session_locks:
        self._session_locks[sid] = asyncio.Lock()
    return self._session_locks[sid]
```

**修复后**:
```python
def _get_session_lock(self, sid: str) -> asyncio.Lock:
    return self._session_locks.setdefault(sid, asyncio.Lock())
```

**影响**: 彻底消除高并发场景下的数据竞争风险。

#### 错误处理安全修复
- 消除所有裸 `except:` 语句，避免意外捕获 `KeyboardInterrupt` 等系统信号
- 为所有异常处理添加适当的日志记录
- 实现多层降级策略确保服务可用性
